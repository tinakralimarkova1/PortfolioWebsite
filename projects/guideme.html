01 / overview

Building confidence through guided practice

GuideMe is an Android application designed to help older adults build confidence using smartphones. Instead of interacting with real apps, users learn through realistic simulations that provide guided, step-by-step lessons, visual cues, and voice instructions in a safe environment. The project focused on improving digital literacy while reducing the anxiety many seniors experience when learning new technology.

03 / key features

Key Features

  • Interactive simulations of Camera, Phone, and Wi-Fi Settings apps
  • 45 guided lessons and practice activities
  • Text-to-Speech guidance with visual highlights
  • User progress tracking and personalized lesson recommendations
  • Natural language search powered by an integrated TensorFlow Lite model
